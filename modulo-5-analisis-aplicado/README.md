# Módulo 5 — Análisis Aplicado

**Semanas 9 y 10 · Clases 17 a 20**

## ¿Por qué este módulo?

Hasta acá aprendiste las herramientas. Ahora las aplicás a problemas reales de negocio. Las empresas no contratan analistas para que hagan gráficos bonitos — los contratan para que respondan preguntas que impactan en decisiones concretas.

En este módulo vas a trabajar con KPIs reales (conversión, churn, ticket promedio), vas a ver casos aplicados a e-commerce y marketing, y vas a lanzar tu proyecto final.
