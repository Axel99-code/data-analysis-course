# Datasets Recomendados

Fuentes de datos para practicar y para el proyecto final.

---

## Para empezar (limpios y pequeños)

- [Iris Dataset](https://archive.ics.uci.edu/ml/datasets/iris) — Clásico dataset de flores. Ideal para estadística básica.
- [Titanic](https://www.kaggle.com/c/titanic/data) — Datos de pasajeros. Perfecto para limpieza y análisis.

## Datos de negocio (más realistas)

- [Superstore Sales](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final) — Ventas de una empresa retail. Ideal para KPIs.
- [E-commerce Dataset](https://www.kaggle.com/datasets/carrie1/ecommerce-data) — Transacciones online. Bueno para análisis de clientes.

## Datos locales (Argentina)

- [datos.gob.ar](https://datos.gob.ar/) — Portal oficial de datos abiertos del gobierno argentino.
- [Buenos Aires Data](https://data.buenosaires.gob.ar/) — Datos abiertos de la Ciudad de Buenos Aires.

## Plataformas generales

- [Kaggle](https://www.kaggle.com/datasets) — La mayor comunidad de datasets del mundo.
- [Google Dataset Search](https://datasetsearch.research.google.com/)
