# Glosario

Términos clave del curso ordenados alfabéticamente.

---

**DataFrame** — Estructura de datos tabular de Pandas, similar a una hoja de cálculo, con filas y columnas etiquetadas.

**EDA (Exploratory Data Analysis)** — Proceso de exploración inicial de un dataset para entender su estructura, distribuciones y relaciones antes de aplicar modelos o sacar conclusiones.

**KPI (Key Performance Indicator)** — Métrica clave que una empresa usa para medir el progreso hacia un objetivo de negocio.

**Outlier** — Valor atípico en un dataset que se aleja significativamente del resto de los datos.

**Serie** — Estructura unidimensional de Pandas. Es una columna de un DataFrame.
