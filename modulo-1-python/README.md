# Módulo 1 — Fundamentos de Python

**Semanas 1 y 2 · Clases 1 a 4**

## ¿Por qué empezamos acá?

Antes de analizar datos, necesitás un lenguaje para hablarle a la máquina. Python es ese lenguaje — y la buena noticia es que su sintaxis es lo suficientemente cercana al inglés como para que en pocas clases ya puedas hacer cosas útiles.

Este módulo no es un curso de programación completo. Es lo mínimo necesario para que puedas manipular datos reales en el Módulo 2. Cada concepto que aparece acá va a volver a aparecer después, aplicado a algo concreto.
