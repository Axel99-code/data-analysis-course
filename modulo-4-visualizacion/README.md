# Módulo 4 — Visualización de Datos y Storytelling

**Semanas 7 y 8 · Clases 13 a 16**

## ¿Por qué este módulo?

Un análisis que no se puede comunicar no sirve. Este módulo te enseña a convertir números en gráficos que cuentan una historia — y más importante, a elegir el gráfico correcto para cada pregunta.

Vas a trabajar con Seaborn para visualización estadística y con Matplotlib para personalización. Pero la parte más importante de este módulo no es técnica: es aprender a estructurar un análisis para que una persona de negocios pueda tomar decisiones con él.

- Crear visualizaciones estadísticas con Seaborn

- Elegir el tipo de gráfico correcto según el mensaje
